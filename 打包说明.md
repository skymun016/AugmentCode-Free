# AugmentCode-Free 打包说明

## 概述

本项目提供了完整的跨平台打包解决方案，可以将Python应用打包为Windows和macOS下的可执行文件。

## 打包方式

### 方式一：自动打包脚本（推荐）

#### Windows系统：
1. 双击运行 `build.bat`
2. 脚本会自动安装依赖并开始打包
3. 打包完成后在 `AugmentCode-Free-windows/` 目录找到可执行文件

#### macOS/Linux系统：
1. 在终端中运行：`./build.sh`
2. 脚本会自动安装依赖并开始打包
3. 打包完成后在 `AugmentCode-Free-darwin/` 或 `AugmentCode-Free-linux/` 目录找到可执行文件

### 方式二：手动打包

1. 安装打包依赖：
   ```bash
   pip install -r build_requirements.txt
   ```

2. 运行打包脚本：
   ```bash
   python build.py
   ```

## 打包结果

打包完成后会生成以下文件：

### Windows:
- `dist/AugmentCode-Free.exe` - 可执行文件
- `AugmentCode-Free-windows/` - 分发包目录
  - `AugmentCode-Free.exe` - 主程序
  - `README.md` - 项目说明
  - `使用说明.txt` - 使用指南

### macOS:
- `dist/AugmentCode-Free.app` - 应用程序包
- `AugmentCode-Free-darwin/` - 分发包目录
  - `AugmentCode-Free.app` - 主程序
  - `README.md` - 项目说明
  - `使用说明.txt` - 使用指南

### Linux:
- `dist/AugmentCode-Free` - 可执行文件
- `AugmentCode-Free-linux/` - 分发包目录
  - `AugmentCode-Free` - 主程序
  - `README.md` - 项目说明
  - `使用说明.txt` - 使用指南

## 技术细节

### 打包工具
- 使用 **PyInstaller** 进行打包
- 支持单文件打包，包含所有依赖
- 自动处理隐藏导入和数据文件

### 包含的文件
- 所有Python源代码
- `augment_tools_core` 模块
- `README.md` 文档
- 必要的系统库和依赖

### 优化特性
- UPX压缩减小文件大小
- 去除调试信息
- 隐藏控制台窗口（GUI模式）

## 故障排除

### 常见问题

1. **Python版本问题**
   - 确保使用Python 3.7或更高版本
   - 检查PATH环境变量设置

2. **依赖安装失败**
   - 尝试升级pip: `pip install --upgrade pip`
   - 使用国内镜像: `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r build_requirements.txt`

3. **打包失败**
   - 检查磁盘空间是否充足
   - 确保没有杀毒软件干扰
   - 尝试以管理员权限运行

4. **macOS权限问题**
   - 运行前确保脚本有执行权限: `chmod +x build.sh`
   - 如遇Gatekeeper问题，在系统偏好设置中允许运行

### 日志查看

打包过程中的详细日志会显示在控制台中，如遇问题请查看错误信息进行排查。

## 自定义配置

- 修改应用名称
- 添加图标文件
- 调整包含的文件和目录
- 设置版本信息

## 支持的平台

- ✅ Windows 10/11 (x64)
- ✅ macOS 10.14+ (Intel/Apple Silicon)
- ✅ Linux (Ubuntu 18.04+, CentOS 7+)

## 注意事项

1. 首次打包可能需要较长时间下载依赖
2. 生成的可执行文件较大（约50-100MB），这是正常现象
3. 建议在目标平台上进行测试确保兼容性
4. 打包后的程序启动可能比源码运行稍慢，这是正常现象
